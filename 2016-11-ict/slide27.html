OpenAjax Accessibility Task Force

Purpose

  • Update and develop new rules:
    • Changes in accessibility HTML5 and ARIA techniques
    • Changes in browser and AT implementation
    • Support new technologies like Scalable Vector Graphics (SVG)
    • Improved messaging for summary, definition or purpose
  • Define rulesets:
    • Custom rulesets
    • Required rules
    • Recommended rules
  • Typically meets third Thursday of each month at 2:00pm Central Time
  • Contact Jon Gunderson to join the group
  • More information on OpenAjax Accessibility Task Force
