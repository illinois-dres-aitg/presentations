Landmark Regions: Resources and Testing Tools

Bypass Block Script: SkipTo

  • SkipTo script adds a "Skip to Content" menu button to the web page for navigating high level landamrks and headings. It is the modern way to implement that WCAG Bypass blocks requirement and relaces "Skip To Main" link technqiue.

Browser Extension: SkipTo

  • SkipTo Landmarks & Headings browser extensions for Firefox and Chrome is an assistive technology to provide keyboard access to the headings and landamrks of a web page.
