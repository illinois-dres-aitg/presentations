Major Changes in SkipTo Version 5.x

  • Use scripted shortcut key to improve consistency between browsers:
    • Alt-0 (Windows/Linux)
    • Option-0 (macOS)
  • Support custom web components.
  • Button switches to SkipTo icon on small screens.
  • Simplify operation and configuration options.
  • Color theme includes a new way to automatically apply a theme using:
    • hostname selector
    • path selector
