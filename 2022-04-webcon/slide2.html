Keyboard Support: Sequential Navigation and Cursor Keys

  • Using the tab key to sequentially navigate to make sure all the links, form controls and any other interactive elements on the page can be accessed with the keyboard.
  • Pressing Space or Enter can activate a link, form control or other interactive elements.
  • Some interactive elememnts like menus, toolbars, spin buttons and sliders support cursor keys to move between items or change the value of the controls.
  • Menus can also support character keys to move between items that start with that character
  • Modal dialog boxes must constrain keyboard navigation to the controls in the dialog box.
  • Keyboard focus must always be on visible elements.
  • Bypass Blocks feature: Either a SkipTo link or "Skip To Main" link.
