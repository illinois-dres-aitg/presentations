Keyboard Support: Testing

  • Move your mouse away from your work station or cover your track pad to allow your attention to be only on using the keyboard. Note: Recommend using Firefox since it's default focus styling is less apparent than Chrome or Safari.
  • Use the tab key to navigate to links, form controls and other interactive content on the page.
  • Questions to ask your self:
    • Can I tell where keyboard focus is on the page?
    • Can I activate controls using the Space or Enter keys? DO they perform the same action?
    • Does the navigation of controls seem predictable.
    • Does focus ever disappear and then magically reapear? If you answer yes, this means there is content being obscured and this is not good for accessibility. Only visible elements should have focus.
    • Can I use the cursor keys to navigate or change the state of some types of interactive elements.
