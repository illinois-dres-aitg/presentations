Keyboard Testing

Keyboard Testing Commands
Key Expected Behaviors for Accessibility
Tab
Shift-Tab
  • Use the "TAB" key to move focus between links, form controls and custom widgets on a web page.
  • Verify you can see which element has the focus.
  • If it is a link read the link text to make sure it describes the target of the link.
Space
Return/Enter
  • "Space" or "Return/Enter" key activates links and buttons, or chages the state of checkboxes or radio buttons.
Cursor Keys
  • For menus of links or actions do the cursor keys allow you to move keyboard focus between items.
Skip to Main
  • Is there a way to "skip to" main content?
