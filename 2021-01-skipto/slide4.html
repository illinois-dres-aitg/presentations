Which disabilities benefit from SC 2.4.1 Bypass Blocks?

  • Keyboard only users: People with physical impairments that limit use of the mouse or users of alternative input devices.
  • Screen reader users: Originally the primary beneficiary of the requirement. Improvements in web design and screen reader supporting heading and landmark navigation the feature is less important. For example h1 elements and/or main landmarks are typically used for the main content and screen have built-in commands to navigate to these locations.
