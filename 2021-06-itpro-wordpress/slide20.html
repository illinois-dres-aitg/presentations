Illinois Theme for WordPress

Similar Features of AITG Website

  • Landmark structure.
  • Page title and h1 synchronization.
  • Color contrast exceeds WCAG AAA requirements.
  • Use of a ARIA disclosure menu pattern for banner menu.
  • Uses SkipTo script for Bypass blocks, but configuration differences:
    • Initially menu button is hidden (e.g. "popup" option), instead of the default visible.
    • Currently not using the CDN version of script.
    • Currently not using the "Illinois" color theme option.
  • Website search feature
  • High contrast support, but instead of using SVG graphics it used character codes for magnifying glass and menu cheverons for search and banner menu supports operating system high contrast modes.
