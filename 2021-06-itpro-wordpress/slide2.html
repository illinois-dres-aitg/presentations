Template: Accessibility Requirements

Success Criterion Level Requirement
1.3.1 Info and Relationships A Use of nav element to create a navigation landmark region to identify the menu of links.
1.3.2 Meaningful Sequence A The template implements meaningful sequence by identifying the header, main and footer information in a logical order.
1.4.10 Reflow AA Using a responsive design and media queries our template works on very narrow renderings.
2.4.1 Bypass Blocks A Using SkipTo script to implement bypass requirement.
2.4.2 Page Titled A The php script automatically updates the title element with the h1 element content.
2.4.5 Multiple Ways AA The banner navigation menu and the search function provide two ways to find and navigate to pages in the website.
2.4.6 Headings and Labels AA The landmarks are properly labelled and headings are used in a structural way.
3.1.1 Language of Page A Include the lang attribute on the html element.
