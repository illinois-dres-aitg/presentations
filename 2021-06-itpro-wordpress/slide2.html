Landmark, Headings and Title: Accessibility Requirements

Landmarks

  • All rendered content in a landmark
  • Properly identify the following landmark regions:
    • banner
    • contentinfo
    • main
    • navigation
    • search

Headings

  • h1: Is the first heading in the main landmark to identify the main content of the page and contains the page title.
  • h2-h3: Identify the sections and sub-sections of the main content.

Title

  • title: Contains the title of the page first and then uses a separator character (.e.g. - ) and then the name of the website.
  • h1: Contains a title for the page content.
