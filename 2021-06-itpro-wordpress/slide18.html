Bypass Blocks: Accessibility Requirements

  • 2.4.1: Bypass Blocks requires a means for keyboard only used to navigate to the main content of the page, essentially bypassing navigational and other links that appear before the main content.
  • Typically bypass blocks is achieved with a "Skip To Main" link that is initially invisible, but appears when it receives keyboard focus.
