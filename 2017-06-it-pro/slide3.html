Definitions

Website Fail
Number of websites that have at least one failure on any page in the website.
Website Pass
Number of websites that all of the pages pass the rule.
Percent Pass
The number of websites that pass divided by the total number of websites the rule applied to.
Implementation Score
An indicator of how pervasive the implementation of a rule requirement.
For element level rules, this indicates the number of elements that passed the rule divided by the number ot total elements the rule applied to.
For page level rules the number of pages that passed in a website versus the toal number of pages the rule applied to.
