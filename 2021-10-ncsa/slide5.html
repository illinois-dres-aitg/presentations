Comments on Accessibility of NCSA Website

  • NCSA Home Page
  • No active keyboard focus styling, website is relying on the default browser focus styling, which is different in each browser and often difficult to see.
  • Misuse of Accessible Rich Internet Applications (ARIA) markup, remember "no ARIA is better than bad ARIA", examples of issues:
    • link role in page banner, probably not needed, but no keyboard support and poor accessible name (e.g. essentially all the text content in the banner).
    • menubar and menuitem roles do not describe the actual keyboard behavior of the page, actual keyboard navigation seem very unpredictable to me.
