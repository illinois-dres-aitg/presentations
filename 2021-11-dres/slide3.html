History of IT Accessibility at DRES

Selected Highlights

  • 1995: Assistive Technology lab at DRES with hours restricted to building hours.
  • 1997: Created assistive technology rooms in 4 libraries across campus available during library hours.
  • 2003: Accessible publishing Wizard for Microsoft Word and Powerpoint.
  • 2004: Functional Accessibility Evaluator (FAE) 1.0 released for free use by anyone.
  • 2010: FAE 1.1 released for free use by anyone.
  • 2010: AInspector Sidebar for Firefox released.
  • 2014: FAE 2.0 released for free use by anyone and made open source.
  • 2016: AInspector WCAG for Firefox released.
  • 2017: First global accessibility awareness day event at Illinois.
  • 2018: A11lyFirst enhancements to CKEditor released to help people create accessible web content.
  • 2020: Relased SkipTo script for websites to implement WCAG requirement for ByPass blocks.
  • 2021: Released SkipTo Browser Extension for Firefox and Chrome to provide keyboard navigation to landmarks and headings.
