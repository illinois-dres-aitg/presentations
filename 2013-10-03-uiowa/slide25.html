HTML5 Video Element Accessibility: Example

  • video element supports captioning through the track element
  • WebVTT timed text format
  • Example of an accessible media player used in coursera.org lectures
  • Example that uses HTML5 specifications, but not supported in browsers

Video Element Example

  <video controls">
    <source src="media/sintel_trailer-480p.mp4" type="video/mp4">
    <source src="media/sintel_trailer-480p.ogv" type="video/ogg"/>
    <track kind="subtitles" src="media/sintel_trailer-480p_en.vtt" srclang="en" label="English for the Hard of Hearing">
  </video>
