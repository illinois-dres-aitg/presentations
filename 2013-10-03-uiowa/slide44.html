Third Step: Use ARIA to improve the accessibility of Forms and Links

  • Use aria-required='true' attribute on controls that are required
  • Use aria-invalid='true' attribute on controls with invalid content or formatting, and use aria-describedby to point to a message on the problem and how to fix it
  • When the HTML label element is insufficent for for labeling controls, use aria-labelledby and aria-label to label controls
  • Use aria-describedby to provide context to links with the same or similar link text
  • Use HTML5 required attribute instead of aria-required if you are supporting on HTML5 browers
