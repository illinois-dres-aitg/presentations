aria-describedby to Provide Link Context

  • WCAG 2.0 Success Criterion 2.4.4 Link Purpose (In Context): The purpose of each link can be determined from the link text alone or from the link text together with its programmatically determined link context, except where the purpose of the link would be ambiguous to users in general. (Level A)
  • aria-describedby attribute can point to the appropriate context
