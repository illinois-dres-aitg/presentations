About the CITES Player

The CITES video player is a modification of the jPlayer HTML5 audio and video player created and released by happyworm. The CITES modification is a minor addition to include the ability to display captions. These modifications were made to jPlayer release 2.1.0.

The player has two limitations regarding captions:

  1. Only xml formatted caption files are allowed.
  2. An Access-Control-Allow-Origin HTTP response header must be added to allow the ajax get request to succeed for the subtitle file (see enable-cors.org for more information about cross-domain response headers). This is not necessary if the video and caption files are stored on the webserver (i.e. same domain).
