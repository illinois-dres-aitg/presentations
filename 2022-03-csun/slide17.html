SkipTo Summary

  • SkipTo script and browser extensions are open source resources and are free to use or modify to your own needs.
  • Use existing page landmark and heading structure to generate a menu for orientation and navigation.
  • SkipTo is a major advancement in implementing the WCAG 2.4.1 Bypass Blocks of Content requirement for web pages.
  • SkipTo Landmarks & Headings browser extensions for Chrome and Firefox implements a long standing browser accessibility requirement for keyboard navigation to structural content.
