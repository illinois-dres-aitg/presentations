Goals of SkipTo script and browser extensions

  • Improve keyboard accessibility to web page content. Basically give all users some of the header and landmark navigation features available in screen readers.
  • Allow authors to identify more than one important navigational point within web resources.
  • Simplify implementation of WCAG 2.4.1 Bypass Blocks of Content requirement.
  • Make header and landmark region information available to everyone to promote a better understanding of accessibility.
