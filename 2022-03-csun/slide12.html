SkipTo: Installing and Configuration

  • Include reference to skipto.js script in the page template:
    • CDN Link: https://paypal.github.io/skipto/downloads/js/skipto.min.js
    • Reference to a local file system copy of skipto.js
  • To change default settings use the SkipToConfig object.
    • Menu button is visible on load or pops up with focus.
    • Landmarks and headings shown in menu.
    • Colors used for button and menus.
    • L ist of all configuration options.

Example Configuration

  <script src="https://paypal.github.io/skipto/downloads/js/skipto.min.js"></script>
  <script>
      var SkipToConfig =  {
          'settings': {
              'skipTo': {
                  landmarks: 'main, [role="main"], [role="search"], nav',
                  headings: 'main h1, main h2',
                  colorTheme: 'illinois',
                  customClass: 'aitg'
              }
          }
      };
  </script>
