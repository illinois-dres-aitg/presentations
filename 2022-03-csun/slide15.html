SkipTo Landmarks & Headings Browser Extensions for Chrome and Firefox

  • Open source and free to use.
  • Adds the SkipTo functionality to the Chrome and Firefox browsers.
  • Provides access to the landmark and heading structure of web pages for anyone to benefit.
  • Implements a W3C User Agent Accessibility Guidelines requirement for:
    • 1.9.1 Outline View
    • 2.2.2 Sequential Navigation Between Landmarks:
  • Keyboard shortcuts:
    • Windows: Alt+2
    • macOS: Option+2
